# **CAR GAME**

### About the game;
- Various car options.
- Customizable theme and menu arrangements.
- Different speeds and types of obstacles for each level.
- Tree images specific to levels.
- Menu sound effects and in-game radio settings.
- Obstacles and different behavior patterns that become more difficult as you level up.
